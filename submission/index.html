📦 사용법

3초 컷 시작 가이드

설치랄 것도 없습니다.

1

실행

실행.vbs 더블클릭. cmd 창 없이 조용히 백그라운드 + 트레이 아이콘으로 상주합니다.

2

켜기

Ctrl + 0 → 미니창이 뜨고 시청자순 목록이 나옵니다. 보고 싶은 채널 클릭.

3

조작

화면을 드래그해 이동, 가장자리로 크기 조절, 우클릭으로 뒤로가기·종료.

4

숨기기

Ctrl+0 또는 Esc, 혹은 카메라가 움직임을 감지하면 자동으로. 다시 켜면 보던 채널 그대로.

5

설정

트레이 아이콘 우클릭 → 옵션… 에서 모션 감지 민감도 조절. 끄고 싶으면 체크 해제.
